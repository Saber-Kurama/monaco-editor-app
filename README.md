## Monoca Editor App

本仓库用于演示[Monoca Editor](https://microsoft.github.io/monaco-editor/)的用法

## 运行

- node 版本：14.18.1，npm 版本：6.14.15。如果发现安装的依赖不对，可以对齐下 node 和 npm 版本
- 运行项目：
  - npm install
  - npm run start
